# ProductSelector – Sistema Web de Ventas Personalizadas

ProductSelector es un sistema web desarrollado como proyecto académico para apoyar a vendedores en la atención personalizada de clientes, permitiendo seleccionar PCs o laptops según las necesidades del cliente.

## Funcionalidades
- Selección de tipo de equipo: PC o Laptop
- Flujo personalizado según la elección del cliente
- Configuración de PC por componentes
- Carrito de compras con cálculo de precios
- Selección de laptops según carrera profesional
- Filtros por procesador (Intel Core / AMD Ryzen)
- Visualización de características y compatibilidad
- Sistema de autenticación (login / logout)
- Panel de administración para gestión de productos
- Conexión a base de datos MySQL

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Base de datos
El proyecto incluye un archivo SQL de ejemplo para la creación de la base de datos.

## Nota
Este proyecto es una adaptación académica y no representa a una empresa real.
